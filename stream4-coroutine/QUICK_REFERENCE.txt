┌──────────────────────────────────────────────────────────────────────────────┐
│ WHAT EACH COMPONENT DOES                                                    │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ 1. CFG Construction                                                          │
│    • Builds control flow graph from AST                                     │
│    • Creates basic blocks and edges                                         │
│    • Tracks variable uses and definitions                                   │
│    • Identifies suspend points                                              │
│                                                                              │
│ 2. Liveness Analysis                                                         │
│    • Computes gen/kill sets per block                                       │
│    • Performs backward dataflow iteration                                   │
│    • Determines which variables are live at each point                      │
│    • Converges in 2-5 iterations                                            │
│                                                                              │
│ 3. Frame Layout                                                              │
│    • Creates state struct per suspend point                                 │
│    • Each state contains only live variables                                │
│    • Computes sizes, alignments, offsets                                    │
│    • Generates tagged union layout                                          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────────────────┐
│ KEY ALGORITHMS                                                               │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ CFG Construction: Recursive Descent                                          │
│   Time: O(n)  Space: O(b+e)                                                  │
│   • Traverse AST, create blocks and edges                                   │
│   • Split blocks at control flow boundaries                                 │
│                                                                              │
│ Liveness Analysis: Backward Dataflow                                         │
│   Time: O(b×v×i)  Space: O(b×v)  [i < 5 typically]                          │
│   • GEN[B] = vars used before def                                           │
│   • KILL[B] = vars defined                                                  │
│   • LIVE_OUT[B] = ∪ LIVE_IN[successors]                                     │
│   • LIVE_IN[B] = GEN[B] ∪ (LIVE_OUT[B] - KILL[B])                           │
│                                                                              │
│ Frame Layout: Tagged Union                                                   │
│   Time: O(s×f)  Space: O(s×f)                                                │
│   • Create state struct per suspend point                                   │
│   • Union size = max(all state sizes)                                       │
│   • Frame = tag + aligned union                                             │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
